# git

Pi extension: structured git tools for staging, committing, and history.

## Install

```sh
pnpm add git
```

Exposes tools: `git_status`, `git_add`, `git_rm`, `git_mv`, `git_diff`, `git_restore`, `git_commit`, `git_log`, `git_push`.

## Development

```sh
pnpm install
pnpm run check  # typecheck + lint + test
```
