# ripgrep

Pi extension: structured wrapper around `rg` for file content search.

## Install

```sh
pnpm add ripgrep
```

Requires ripgrep to be installed:

```sh
brew install ripgrep  # macOS
apt install ripgrep  # Linux
cargo install ripgrep # via Rust
```

See [ripgrep](https://github.com/BurntSushi/ripgrep) for full documentation.

## Development

```sh
pnpm install
pnpm run check  # typecheck + lint + test
```
