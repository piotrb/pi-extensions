# bfs

Pi extension: structured wrapper around the `bfs` breadth-first file finder.

## Install

```sh
pnpm add bfs
```

## Usage

Install bfs first:

```sh
brew install bfs  # macOS
apt install bfs    # Linux
```

See [bfs](https://tavianator.com/projects/bfs.html) for full documentation.

## Development

```sh
pnpm install
pnpm run check  # typecheck + lint + test
```
